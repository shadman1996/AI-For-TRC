Make Me Admin – Temporary Administrator Access

The Make Me Admin tool allows approved users to temporarily receive administrator rights on their computer. This access is intended for tasks such as installing software or changing system settings that require elevated permissions.

Administrator access is time‑limited and will be removed automatically.

Windows Instructions 

Click the Start menu, search for Make Me Admin, and run the application. 

In the Make Me Admin window, click Grant Me Administrator Rights. 

The application will close, and a Windows notification will appear in the lower-right corner confirming that administrative privileges have been granted. 

You will have admin rights for 10 minutes. After that, you will automatically be removed from the Administrator group. However, any applications you launched with elevated access will continue to run with those privileges. 

You will receive a notification when your administrator rights have been removed. 

To manually remove admin rights before the 10-minute window ends, run Make Me Admin again and click Remove My Administrator Rights. 

Once you have admin rights, you may still be prompted to enter your credentials when installing software or changing system settings: 

Username: smsu.edu\StarID (e.g., smsu.edu\ab1234cd) 

Password: Your StarID password 

Important: When installing software, always right-click the installer and select Run as administrator. 
 

macOS Instructions

Open the Self-Service+ application. 

Search for Make Me Admin in Self-Service+. 

Click Run under Make Me Admin. 

A warning telling you "You now have administrative rights for 30 minutes. DO NOT ABUSE THIS PRIVILEGE...” will appear, click “Make me an admin, please” 

Wait for the buffer to finish then you will have admin rights for 30 minutes.
